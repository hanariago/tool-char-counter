# 글자 수 세기

텍스트를 붙여넣으면 글자 수를 즉시 카운트.

🔗 **바로 쓰기**: https://hanariago.github.io/tool-char-counter

## 기능
- 공백 포함 / 공백 제외 글자 수
- 줄 수 · 단어 수 · 문장 수
- 실시간 카운트 (입력 즉시 반영)
- 초기화 버튼

## 기술 스택
- 순수 HTML/CSS/JS (외부 라이브러리 없음)
- 서버 전송 없음 (브라우저에서만 동작)
